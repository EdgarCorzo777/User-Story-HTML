# Edgar Corzo - Personal Portfolio

A personal portfolio website built with HTML, CSS and JavaScript as part of my Software Development learning journey at Riwi.

## 📄 Pages

- **index.html** — Main page with About me, Projects and Contact sections
- **pets.html** — Pets gallery page

## 🛠️ Technologies

- HTML5
- CSS3 (Flexbox, Grid, Media Queries, CSS Variables)
- JavaScript (DOM manipulation)

## ✨ Features

- Responsive design for mobile, tablet and desktop
- Sticky header with smooth navigation links
- Projects section organized with CSS Grid
- Interactive "Read more / Read less" button
- Contact form with show/hide toggle
- Pets gallery with uniform image layout
- Optimized CSS with global variables and minified version

## 📁 Project Structure

```
portafolio-sem2/
├── index.html
├── pets.html
├── style.css
├── style.min.css
└── script.js
```

## 🚀 How to run

Just open `index.html` in your browser — no installation required.
